Programmierung

Für die Programmierung des Gadgets wird die "Arduino IDE" benötigt. Diese kann hier heruntergeladen werden und muss anschließend installiert werden. Nun wird der Boardverwalter geöffnet (Werkzeuge>Board>Boardverwalter) und dort das Paket "esp32" installiert.

Das Programm kann hier heruntergeladen werden: TODo
MultiMonG.zip

Anpassung des WLAN

Da für den Code WLAN benötigt wird, müssen in den folgenden zwei Zeilen noch der passende WLAN-Router mit zugehörigem Passwort eingestellt werden. Zu Testzwecken kann auch der Hotspot vom Handy verwendet werden.

In der oberen Zeile muss für „WLAN-Name hier eingeben“ der Name des WLAN-Netzwerkes eingegeben werden. In der zweiten Zeile muss für “WLAN-Passwort hier eingeben“ das passende WLAN-Passwort eingetragen werden. Im Code findet man dies in den Zeilen 12 und 13.

Anpassung des Ortes für die Außentemperatur

Um die Außentemperatur aus dem Internet abrufen zu können, muss sich zunächst auf einer Website registriert werden. Diese ist unter folgendem Link zu finden:
Weather API - OpenWeatherMap

Über „sign in“ oben rechts und anschließend über „create an account“ sollte sich folgende Seite geöffnet haben.

Hier muss nun ein Account erstellt werden.

Nach dem Einloggen kann über den Username, der nun anstelle von Login in der oberen Zeile steht, „My API keys“ ausgewählt werden.

Der API key muss nun in den Zeilen 22-24 im Code zwischen die “ “ eingefügt werden. 


          string openWeatherMapApiKey = ""; //ApiKey aus der Webseite einfügen
          string city = "Karlsruhe"; //Standort eingeben
          string country Code = "DE"; //Land in Kurzschreibweise
          string jsonBuffer;

Unter „string city“ lässt sich die Stadt ändern, für die die Temperatur und Luftfeuchtigkeit angezeigt werden soll. Dafür muss einfach „Karlsruhe“ für die gewünschte Stadt ersetzt werden. Sollte eine Stadt außerhalb von Deutschland gewählt werden, muss in der Zeile darunter für „DE“ das passende Land in Kurzschreibweise ersetzt werden.

Libraries

Nun müssen noch einige Libraries installiert werden. Dafür muss in der linken Seitenleiste das Büchersymbol ausgewählt werden. Es sollte sich nun der Library Manager öffnen. Insgesamt müssen fünf Libraries installiert werden.

Wie in dem Bild zu sehen ist, muss einmal in der Suchleiste nach FastLED gesucht werden und die vorgeschlagene Library anschließend installiert werden.

Die zweite Library, die benötigt wird, kann unter Esp32time gefunden werden.

Die dritte Library, die benötigt wird, kann unter ESP-DASH gefunden werden.

Die vorletzte Library die benötigt wird, kann unter Arduino JSON gefunden werden.

Die Letze Bibliothek für den DHT11 Sensor (Temperatur und Luftfeuchtigkeit)

Code hochladen

Ist dies erledigt muss nun der Code auf den Arduino hochgeladen werden. Hierfür muss zunächst der Arduino über den USB-C-Anschluss mit dem Laptop verbunden werden. Anschließend muss nun der richtige Port und das richtige Board eingestellt werden. In der Arduino Software kann dies in der oberen Leiste unter „select Board“ und anschließend unter „Select other board und port“ eingestellt werden.

Es sollte sich nun folgendes Fenster geöffnet haben:

Wie in dem Bild zu sehen ist, muss in der Suchleiste nach „Arduino Nano ESP32“ gesucht werden. Hierbei ist es wichtig darauf zu achten, dass das Board „Arduino Nano ESP32-esp32“ ausgewählt wird. Unter Ports wird einfach der angezeigte Port ausgewählt.

Nun kann der Code über den Pfeil auf türkisem Hintergrund hochgeladen werden.
