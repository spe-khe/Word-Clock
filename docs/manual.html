Word-Clock

Word-Clock Gadget der SPE Karlsruhe

Work in Progress

Bedienung der Word-Clock

Sobald die Wordclock an eine Stromquelle (Laptop oder 15w Netzteil) angeschlossen wird, startet die Wordclock ein Netzwerk namens "SPE-Wordclock".Verbindet man sich mit dem Netzwerk, so wird man automatisch auf eine Website geleitet.

Hier kann der Netzwerkname und das Netzwerkpasswort eingetragen werden, mit dem sich die Uhr verbinden soll. Zusätzlich kann der API Key von OpenWeatherMap und die Postleitzahl des Standorts eingetragen werden. Diese wird benötigt, um die passende Temperatur und Luftfeuchtigkeit abzurufen

Offline Mode

Solange die Wordclock nicht verbunden ist befindet sie sich im "Offline Mode". Hier kann die Uhrzeit per Hand eingestellt werden. Dazu hält man den "Temperaturtaster" gedrückt, bis die Anzeige blinkt. Mit Dem "Farb- und Helligkeitstaster" können nun die Stunden und Minuten angepasst werden. Wenn die Uhrzeit richtig eingestellt ist kann mit erneuten Drücken des "Temperaturtasters" die Uhrzeit gespeichert werden.

Online Mode

Ist die Uhr mit dem Netzwerk verbunden befindet sie sich im "Online Mode" (Der Buchstabe "o" leuchtet in der unteren rechten Ecke). Die Uhrzeit ist automatisch richtig eingestellt.

Temperaturtaster

Mit dem "Temperaturtaster" kann man sich durch die verschiedenen Temperaturen und Luftfeuchtigkeiten wählen.

  • 1. Temperatur innen (Display zeigt "in" für innen)
  • 2. Luftfeuchtigkeit innen (Display zeigt "in" für innen)
  • 3. Temperatur außen
  • 4. Luftfeuchtigkeit außen

bei negativen Temperaturen zeigt das Display "ne" an

Helligkeitstaster

Mit dem Helligkeitstaster lässt sich die Helligkeit des Displays manuell einstellen. Dafür drückt man den Taster so oft, bis die richtige Helligkeit erreicht ist. Hält man den Taster gedrückt, so schaltet die Uhr in den "Helligkeitsautomatik Modus" (Buchstabe "Z" in der unteren linken Ecke leuchtet). Hier wird die Helligkeit entsprechend der äußeren Helligkeit, die durch den Helligkeitswiderstand gemessen wird, angepasst.

Farbtaster

Mit dem Farbtaster lässt sich die Farbe des Displays einstellen. Durch einmal drücken oder gedrückt halten können die Farben durchgewechselt werden.
